# UpStock

Sistema inteligente de gestão de estoque com interface simples, acesso web e mobile, suporte multiusuário, múltiplos depósitos, dashboards personalizados e integração com marketplaces, desenvolvido para resolver dificuldades reais de controle de estoque identificadas em entrevista com o Mercadinho Delmara Dantas [1].

## Visão geral

O **UpStock** foi idealizado como uma solução para a dificuldade na gestão de estoque, problema levantado a partir de uma entrevista de campo [1]. A proposta do projeto é unir usabilidade, acompanhamento em tempo real e análise de dados com IA para apoiar decisões operacionais [1].

## Problema

Muitos negócios enfrentam falhas no controle de entradas, saídas, níveis mínimos e organização geral do estoque [1]. O projeto surgiu justamente para atacar esse cenário com uma plataforma centralizada e mais acessível [1].

## Solução

O UpStock é um sistema de gestão de estoque inteligente com:

- Interface simples e intuitiva [1]
- IA integrada para análise de dados em tempo real [1]
- Dashboards personalizados [1]
- Integração com marketplaces [1]
- Acesso web e mobile [1]
- Suporte multiusuário [1]
- Suporte a múltiplos depósitos [1]

## Funcionalidades

Com base no protótipo apresentado, o sistema conta com módulos como [1]:

- Visão geral do estoque com indicadores do dia
- Cadastro e gerenciamento de produtos
- Controle de movimentações de entrada e saída
- Relatórios de movimentações e giro de estoque
- Sistema de notificações e alertas
- Gestão de usuários
- Configurações do sistema

## Telas do protótipo

O sistema mostra os seguintes fluxos principais [1]:

- **Dashboard** com resumo operacional, valor total do estoque, capacidade utilizada e ações recomendadas
- **Produtos** com catálogo, pesquisa, filtros, preço, quantidade e status
- **Movimentações** com histórico completo de entradas e saídas
- **Relatórios** com geração e exportação de análises
- **Notificações** com alertas de estoque, registros operacionais e eventos do sistema
